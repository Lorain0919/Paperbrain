# PaperBrain MVP v0.1

## Goal

Convert a scientific paper into a structured Obsidian note.

## Input

Markdown file generated by MarkItDown.

## Output

Structured note containing:

- Background
- Objective
- Methods
- Main Findings
- Relation to My Research
- Keywords

## Workflow

PDF
↓
MarkItDown
↓
Markdown
↓
LLM
↓
Structured Note

## Success Criteria

A student can upload one paper and obtain a readable summary in less than 5 minutes.
