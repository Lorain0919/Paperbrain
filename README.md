# PaperBrain

PaperBrain is an AI-assisted research workflow for scientific literature management.
## Problem
Researchers spend a large amount of time:

* Reading PDFs
* Extracting key findings
* Organizing notes
* Connecting ideas across papers

Most tools store papers, but do not help transform them into structured knowledge.

## Goal

PaperBrain aims to create a workflow that:

1. Converts PDF papers into Markdown
2. Extracts key information using AI
3. Organizes knowledge in Obsidian
4. Builds connections between concepts across papers

## Current Workflow

PDF
↓
MarkItDown
↓
Markdown
↓
AI Summarization
↓
Obsidian Knowledge Base
